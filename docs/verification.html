Home / Verification

Verify scientific artifacts

Strict comparison

rustamps verify --run RUN_COPY --golden GOLDEN_DATASET

Strict mode compares expected artifact presence, keys, dimensions, types, and configured numeric tolerances. It returns nonzero on mismatch.

Completed stage prefix

rustamps verify --run RUN_COPY --golden GOLDEN_DATASET \
  --through-stage 6

The prefix changes the artifact contract as well as the files inspected.

Scientific profile

rustamps --config CONFIG verify \
  --run RUN_COPY --golden GOLDEN_DATASET \
  --profile scientific

Scientific mode permits only configured bounded numeric outliers with hard maximum-error caps. Exact structural keys remain exact, and wrapping applies only to configured cyclic quantities.

Acceptance rule

A successful run command is not a parity claim. Retain the verifier report, configuration, source fingerprint, and performance profile with scientific results.
