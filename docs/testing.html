Native validation gates

Repository checks

cargo fmt --all -- --check
cargo test --workspace --locked
cargo build --release --locked
cargo tree -p rustamps

The production release surface is Cargo-only. There is no Python wheel or source-distribution gate.

Dataset validation

rustamps run --dataset RUN_COPY --start-step 1 --end-step 8
rustamps verify --run RUN_COPY --golden GOLDEN_DATASET

Use fresh upstream inputs when validating scientific changes. A checkpoint-resume test is additional evidence, not a replacement for a fresh solve.

Optional oracle checks

The historical implementation is isolated for developer comparison only:

make oracle-setup
make oracle-test
