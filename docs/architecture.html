One native production boundary

A standalone Rust binary reads and writes the StaMPS-style dataset directory. The production path has no Python extension boundary and does not spawn an external scientific tool.

Crates

CrateResponsibility
rustamps-cliCommands and JSON reports
rustamps-pipelineConfiguration, scheduling, invalidation, and publication
rustamps-coreNative preparation and Stage 1–8 algorithms
rustamps-ioDataset discovery and MATLAB-compatible artifact I/O
rustamps-verifyStrict and scientific artifact comparison

State and publication

Stages 1–5 operate on patches; Stage 5 publishes merged root products; Stages 6–8 operate on merged artifacts. A completion marker is committed only after the entire output bundle succeeds.

Stage 6 writes atomic per-interferogram checkpoints with scientific fingerprints and publishes phuw2.mat only when every solve completes.

Native configuration

runtime:
  backend: native
  stage2_kernel_backend: native
  stage6_solver: native
  cpu_workers: 0
  stage6_grid_scale: 1.0
  stage6_max_flow_passes: 0

Provider and solver fallbacks were removed. Unsupported legacy branches fail before producing substitute artifacts.
