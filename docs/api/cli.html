Standalone Rust command

Production commands

  • rustamps status --dataset PATH
  • rustamps run --dataset PATH [--start-step N] [--end-step N] [--dry-run] [--cpu-workers N]
  • rustamps prep snap --dataset PATH
  • rustamps verify --run PATH --golden PATH [--profile strict|scientific] [--through-stage N]
  • rustamps describe-inputs --stage all
  • rustamps describe-backends

Configuration

Pass --config YAML_OR_JSON before the command. See the native configuration contract.
