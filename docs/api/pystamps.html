pystamps

Historical oracle only

This source-only Python package is retained for reference reproduction. The production package is the root Cargo package and exposes the Rust binary.

Exports

  • __version__: str
  • __all__: list[str] (contains __version__)

Version is imported directly from pystamps._version.

Implementation notes

Module has no functions or classes beyond package metadata. Importing pystamps therefore depends on the generated version module being present in the installed package or source tree.
