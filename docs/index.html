StaMPS processing in native Rust

The installed rustamps command is a Rust binary. It does not load Python, MATLAB, SNAPHU, Triangle, system HDF5, or another executable at runtime.

Install

cargo install --path . --locked
rustamps --help
rustamps describe-backends

Run and verify

rustamps prep snap --dataset DATASET
rustamps run --dataset DATASET --start-step 1 --end-step 8
rustamps verify --run DATASET --golden GOLDEN_DATASET

Use a writable dataset copy. Stage writes are transactional and long Stage 6 solves resume from fingerprinted native checkpoints.

Historical oracle material

The retained historical source implementation is developer-only oracle material. Explicit make oracle-* targets reproduce references; the production binary never imports or executes that source tree.
