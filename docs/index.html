pySTAMPS Documentation

pySTAMPS is a Python-first StaMPS migration runtime for INSAR processing workflows. This site documents installation, architecture, configuration, and the API surface used by the command line and library modules.

Project Snapshot

Project and primary language: Python 3.12+.

Primary goals:

  • Dataset discovery and stage status inspection
  • Pipeline execution for stages 1..8
  • Verification against golden datasets
  • Parity audit tooling and CLI entry points

Key features

  • Hybrid thread/process execution
  • Optional GPU kernels with CuPy fallback
  • Configurable tolerances and stage control
  • Production-friendly static docs output

High-level architecture summary

The runtime is organized as layered Python modules: CLI orchestration, pipeline scheduler, I/O helpers, typed execution, and numerical kernels. Artifacts are produced as .mat files and reused across merged/patch stages.

See Architecture for details and diagram specs.
