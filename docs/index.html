Quick start

Install, inspect, run

conda create -n rustamps -c sirbastiano/label/dev -c conda-forge rustamps=0.3.0
conda run -n rustamps rustamps status --dataset /data/run
conda run -n rustamps rustamps run --dataset /data/run --start-step 1 --end-step 8
conda run -n rustamps rustamps verify --run /data/run --golden /data/golden

Prefer source builds? Install Rust 1.89+, clone the repository, and run cargo install --path . --locked.

Built for long scientific runs

Fast where it should be. Conservative where it matters.

Native boundary

Numerical kernels, MAT I/O, orchestration, and verification stay inside the Rust process.

Atomic publication

Completion markers appear only after an output bundle succeeds; reruns invalidate downstream state.

Stage 6 resume

Per-interferogram checkpoints are fingerprinted and checksummed before reuse.

Fail-closed modes

Unsupported scientific branches stop before writes instead of silently changing the method.

Portable artifacts

Conda packages cover x86_64 and ARM64 across the three desktop operating-system families.

Golden-tree verifier

Compare real, complex, wrapped-phase, sparse, character, NaN, and Inf data with reviewed tolerances.

Historical oracle material: retained Python sources reproduce references for audits but are never part of the production runtime.
