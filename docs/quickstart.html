Run Stages 1–8 natively

1. Install

cargo install --path . --locked
pystamps describe-backends

2. Create a writable run copy

export SOURCE_DATASET=/path/to/source_dataset
export RUN_DATASET=/path/to/run_dataset
cp -a "$SOURCE_DATASET" "$RUN_DATASET"
pystamps status --dataset "$RUN_DATASET"

For an unprepared compatible SNAP export, first run pystamps prep snap --dataset "$RUN_DATASET".

3. Rehearse and execute

pystamps run --dataset "$RUN_DATASET" --start-step 1 --end-step 8 --dry-run
pystamps run --dataset "$RUN_DATASET" --start-step 1 --end-step 8

An interrupted Stage 6 resumes from matching atomic checkpoints. To resume the whole pipeline from existing completion artifacts, use --start-step 0.

4. Verify

pystamps verify \
  --run "$RUN_DATASET" \
  --golden /path/to/reference_dataset

Strict comparison is the default. Use --through-stage N for a completed prefix or --profile scientific with a reviewed tolerance config.

Optional faster Stage 6 grid

pystamps --config configs/stage6-fast.yaml run \
  --dataset "$RUN_DATASET" --start-step 6 --end-step 8

The fast and balanced profiles coarsen the grid but keep the integer-flow solve converged. Validate either profile scientifically against a strict-grid result for the dataset.
