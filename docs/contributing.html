Contributing

Repository setup

git clone https://github.com/sirbastiano/pystamps.git
cd pystamps
python -m venv .venv && source .venv/bin/activate
python -m pip install -e .
python -m pip install -e .[dev]

Code standards

  • Type hints and dataclasses are used across public modules.
  • Prefer explicit exceptions derived from module-specific classes.
  • Keep behavior deterministic where feasible and document inferred behavior in implementation.

PR workflow

  1. Implement small, scoped change.
  2. Run local static checks and project tests.
  3. Include examples or reproducible commands in PR description.
  4. Run parity-related checks when touching stage math.

Branching model

Default model follows short-lived feature branches with one purpose per branch. Merge after passing quality checks and audit gates.
